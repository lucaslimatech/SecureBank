Secure Bank

Interface front-end de um sistema bancário digital, desenvolvida com foco em experiência do usuário, validações e simulação de funcionalidades reais.

Sobre o projeto

O Secure Bank é um projeto de estudo que simula a interface de um banco digital. A aplicação permite ao usuário:

Criar conta
Realizar login
Acessar um dashboard
Visualizar contas e transações
Simular transferências

O objetivo principal é desenvolver habilidades em desenvolvimento Front-End e aplicar conceitos de qualidade de software (QA), como validações, tratamento de erros e melhoria da experiência do usuário.
